# Java

Linguagem multiplataforma (WORA - Write Once, Run Anywhere)

*Linguagem compilada e interpretada*

O que é JVM? Java Virtual Machine

      É uma maquina virtual que interpreta e executa os códigos em qualquer sistema operacional.

**Processo de interpretação**

      Código fonte -> JavaC -> Bytecode -> JVM - Sistema operacional ( Windows, MacOs, Linux)


Para rodar o Java voce precisará instalar o JVM na maquina através do JRE ou JDK


### JRE: Java Runtinme Environment

  Usado apenas para executar aplicações.

  - JVM
  - Bibliotecas
  - Suportes e Recursos

  Usado apenas para executar o programa.


### JDK: Java Development Kit

Usado para executar e desenvolver as aplicações

  - JRE (Com todos os recursos)
  - Java Tools
  - JavaC ( Java Compiler)
  - Debugger
