Adicionando uma imagem

Existe um atributo chamado target, que serve para direcionar para onde deseja ser enviado após o click. Ex: _blank serve para criar uma aba externa com o conteúdo

Adicionando um vídeo

Para adicionar um vídeo use a tag <video> em seguida o <source> juntamente com o src. não esquecer de colocar o parâmetro controls para poder adicionar o player no vídeo e o <poster> para coloar uma imagem na frente do player.
