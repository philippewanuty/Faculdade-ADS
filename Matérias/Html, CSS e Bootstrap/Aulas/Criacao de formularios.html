Aprendizado

Nesta aula aprendi como fazer formulário, e como usar o método input para adquirir dados dos usuário.
Na continuaçao (Aula 06) aprendemos mais a fundo sobre formularios e tipos dados para entrada, como Radio, checkbox, number e tags como <selected> que seria um tipo de formulário de seleção e a tag de agrupamento de formulários <fieldset>, onde necessita de uma tag <legend> para dar nome a esse agrupamento.

  • <label> : Nomeia a informalçao que deverá ter no input
  • <input> : Tag responsável por coletar dados
  • <button> : botão para dar funcionalidade a uma acão
  • <textarea> : Campo para descrever informações
  • <fieldset> : tag de agrupamento de formulários
  • <select> : tag de formulário tipo seleção
  • <legend> : nomeia o agrupamento de formulários no <select>

Tipos de input

  • text: input de texto
  • numeber: input para escolha de numeros
  • radio : input de seleção única
  • checkbox : input de multipla seleção
  • date: input de data onde abre um calendário
  • Email: input de Email que apenas recebe o email da forma correta.

Métodos

  • required: informa que um campo é necessário
  • type="" campo para colocar o tipo de dado solicitado.
  • selected : pré seleciona algo em um input de seleção
  • min/max="" : informa a quantidade mínima e máxima em um input number
  • Selected : deixa pré selecionado os inputs que necessitam de escolha
  • checked : deixa pré selecionado os inputs que necessitam de marcação
