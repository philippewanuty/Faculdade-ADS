# Operadores aritimeticos

**Operadores de Atribuição**

    + Adição
    - Subtração
    : Multiplicação
    / Divisão
    % Módulo (resto de divisão)
    ++ incremento
    -- decremento

**Operadores realacionais**

    = = 1
    = Diferente
    < Menor
    > Maior
    >= Maior ou igual
    <= Menor ou igual

**Dados primitivos**

    byte x = 3;
    short y = 4;
    int z = 5;
    long k = 6;
    float u = 2.0;
    double w = 3.0;
    boolean t = true;
